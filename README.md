# Task_3_Code_Alpha
Banking System (C++)

Description:
This is a simple console-based Banking System written in C++. 
It allows a customer to deposit, withdraw money and view transaction history.

Features:
- Create customer with account
- Deposit money
- Withdraw money (with balance check)
- View current balance
- View transaction history
- Menu-driven interface

Technologies Used:
- C++
- Classes and Objects (OOP)

How It Works:
- User enters name, ID, and account number
- Customer account is created
- User can perform deposit and withdraw operations
- Each transaction is stored and displayed when needed

How to Run:
1. Compile the code:
   g++ Task 3.cpp -o program

2. Run the program:
   ./program

Note:
- Maximum 100 transactions are stored
- No file handling (data is not saved permanently)
- All data resets when program restarts

Purpose:
This project is for learning:
- Object-Oriented Programming (OOP)
- Classes (Customer, Account, Transaction)
- Arrays and basic banking logic
